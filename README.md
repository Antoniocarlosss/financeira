# Minha Vida Financeira

Projeto reorganizado em pastas separadas e preparado para instalar como app/PWA.

## Estrutura

```text
minha-vida-financeira/
├── index.html
├── css/
│   └── styles.css
├── js/
│   ├── app.js
│   └── pwa.js
├── assets/
│   └── icons/
├── manifest.json
├── sw.js
└── _original/
```

## Como usar

1. Abra `index.html` no navegador para testar.
2. Para instalar no celular/computador, publique a pasta online, por exemplo no GitHub Pages ou Netlify.
3. No Android/Chrome ou computador, use o botão `Instalar app` quando aparecer.
4. No iPhone, abra no Safari, toque em Compartilhar e escolha `Adicionar à Tela Inicial`.

## Observação

O app salva os dados no navegador com `localStorage`. Se limpar os dados do navegador, os registros podem ser apagados.
